About Generation Q

This is a tool to help teachers come up with reading comprehension questions by automatically generating questions from text.

Questions are generated from input sentences using sequence-to-sequence learning. The model is a Python implementation of the one described in the paper Learning to Ask: Neural Question Generation for Reading Comprehension, and is built on the wonderful open-source neural machine translation framework OpenNMT.

The model is trained with question-answer pairs from the SQuAD dataset. This web app is built in Python with Flask and hosted on AWS.
